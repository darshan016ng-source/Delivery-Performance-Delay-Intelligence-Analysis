# Delivery Performance & Delay Intelligence Analysis

## Project Overview

The **Delivery Performance & Delay Intelligence Analysis** project is an interactive **Power BI** dashboard designed to analyze delivery operations, rider performance, and delay trends. The dashboard provides actionable insights into operational efficiency by tracking key performance indicators (KPIs), identifying delivery bottlenecks, and uncovering the primary causes of delays.

By leveraging **Power BI**, **Power Query**, and **DAX**, this project enables organizations to monitor delivery performance, improve rider productivity, and support data-driven operational decisions.

## Tech Stack

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel / CSV
* Data Cleaning & Transformation
* Data Visualization

## Project Objectives

* Analyze delivery performance metrics.
* Monitor rider efficiency and productivity.
* Identify the major factors contributing to delivery delays.
* Support operational decision-making through interactive dashboards.

## Dashboard Features

* Delivery Performance Tracking
* Rider Efficiency Analysis
* Delay Pattern Analysis
* Traffic and Weather Delay Insights
* Pickup Delay Monitoring
* KPI Cards and Interactive Filters

## Key Performance Indicators (KPIs)

* Total Deliveries
* On-Time Delivery Rate
* Average Delivery Time
* Delayed Deliveries
* Rider Efficiency Score
* Delay Percentage
* Pickup Delay Rate

## Key Insights

* Traffic congestion was one of the leading causes of delivery delays.
* Weather conditions significantly impacted delivery performance during peak hours.
* Certain pickup locations consistently experienced higher delay rates.
* High-performing riders maintained better on-time delivery performance and overall efficiency.

## Project Highlights
